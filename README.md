# Mood - Jeans Store Website

This is a website that I built for a Jeans sotre. It is based on a goodbrief prompt.


## Tech Stack
  - React
  - Typescript
  - Tailwind Css
  - Framer Motion


## Installation


```
    git clone https://github.com/merkariyas/Mood
    cd Mood
    npm install
    npm run dev
```
